# Code of Conduct

Epicode is an open-source project. We want the community to be respectful, welcoming, and safe for everyone.

## Expected behavior

- Be respectful and constructive.
- Focus on the work, not the person.
- Give feedback in good faith.
- Respect differing opinions and experiences.

## Unacceptable behavior

- Harassment, discrimination, or abusive language
- Personal attacks or public shaming
- Deliberate disruption of discussions or contributions
- Sharing private or sensitive information without permission

## Reporting

If you experience or witness unacceptable behavior, please report it through GitHub issues or security advisories, or contact the maintainer at `sunorme@163.com`.

## Scope

This code of conduct applies to project spaces, issues, pull requests, and discussions.
