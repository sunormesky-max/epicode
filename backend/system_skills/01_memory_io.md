# 记忆智能存取 (Memory IO)

## 概述
系统通过记忆创建、搜索和回忆来管理长期上下文。这个技能定义最基本的记忆操作流程。

## 标准流程
1. 先用 `memory_create` 存入新的重要信息
2. 用 `memory_search` 找到相关历史记忆
3. 用 `memory_recall` 获取带关联展开的上下文

## 何时使用
- 用户提供新事实、偏好、决策或约束
- 需要确认之前是否已经记录过类似信息
- 需要从历史上下文恢复工作状态

## 使用建议
- 内容要简洁，但保留足够上下文
- 标签要稳定一致，便于后续检索
- 重要结论要立即保存，不要等到会话结束
