# 知识图谱导航 (Knowledge Graph Navigation)

## 概述
系统自动将记忆组织成知识图谱，相关记忆会自动聚类。本技能教你如何利用这个图谱发现记忆之间的隐藏关联。

## 使用方法

### 查看图谱
使用 `knowledge_graph` 工具获取图谱数据。

### 利用图谱信息
1. **发现关联**: 搜索结果中可以检查同一组的其他记忆，它们往往相关
2. **识别领域**: 通过标签分布了解知识覆盖的领域
3. **发现缺口**: 如果某个领域记忆很少，说明该领域知识不足
4. **质量感知**: 相关记忆多的领域通常知识更成熟可靠

## 图谱驱动的智能行为
1. 搜索时优先关注记忆丰富的领域（更可靠）
2. 存储新记忆时参考同类记忆的标签模式
3. 发现孤立记忆时主动提示用户补充相关信息
4. 利用关联为用户推荐相关知识："你还提到了..."
