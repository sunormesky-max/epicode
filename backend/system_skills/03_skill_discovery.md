# 技能发现引擎 (Skill Discovery Engine)

## 概述
系统拥有一个包含数百个公共技能的技能库，覆盖多个领域。本技能教你如何发现和使用适合当前场景的技能。

## 技能发现方法

### 自动发现
当你接收到用户请求时，先用 `skill_execute` 搜索相关技能：
```
skill_execute(query="用户请求的关键词", context="当前对话上下文")
```

### 搜索技巧
1. **用领域关键词**: "数据分析"、"代码审查"、"写作"
2. **用问题类型**: "比较"、"分类"、"总结"
3. **组合搜索**: 同时提供query和context提高匹配精度

### 结果解读
`skill_execute` 返回：
- `skill`: 最佳匹配技能（含完整内容）
- `alternatives`: 备选技能（最多3个）
- `total_matched`: 总匹配数

### 执行技能
技能内容是Markdown格式的指导文档。你需要：
1. 阅读技能内容中的步骤和指南
2. 按照技能描述的方法执行任务
3. 执行后用 `feedback_submit` 反馈技能是否有帮助

## 最佳实践
1. 每次处理复杂任务前，先搜索是否有相关技能
2. 如果找到匹配技能，优先按技能指导执行
3. 如果没找到，按自己的能力执行即可
4. 定期反馈技能质量，帮助系统排序优化
