# 质量自控 (Quality Self-Control)

## 概述
系统会自动进行记忆质量控制，但你也可以主动参与。本技能教你如何帮助维护记忆库的健康。

## 你能做的

### 主动提升质量
1. 搜索时发现有用的结果 → `feedback_submit(relevance="highly_relevant")`
2. 发现有过时/错误的信息 → `feedback_submit(relevance="irrelevant")` 降低其排名
3. 存入新版本信息时，在content中注明"替代了关于X的旧信息"

### 避免质量降级
1. 不要重复存储相同信息（先搜索再存）
2. 给记忆足够多的标签（提高可发现性）
3. content要完整准确（不完整的记忆很难被匹配）

### 发现问题记忆
使用 `memory_search` 定期检查：
- 搜索核心概念，看结果是否相关
- 无关内容 → 反馈为irrelevant
- 缺少内容 → 创建新记忆补充

## 健康记忆库的特征
- 搜索命中率较高
- 各领域都有足够的记忆
- 定期有新记忆存入和反馈提交
- 少量重复或矛盾信息（系统会自动处理）
